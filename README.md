# Bridge Website

Professional website for Bridge mentorship app legal documents and landing page.

## 🌐 Live Site
**Production:** https://mybridgementor.com

## 📁 Structure

```
bridge-website/
├── index.html           # Homepage
├── privacy.html         # Privacy Policy
├── terms.html           # Terms of Service
├── safeguarding.html    # Safeguarding Policy
└── README.md           # This file
```

## 🚀 Deployment

This site is automatically deployed via Vercel:
- **Push to main** → Auto-deploy to production
- **Pull request** → Preview deployment

## 📝 Updating Legal Documents

1. Edit the HTML files directly
2. Commit and push to main
3. Vercel auto-deploys within 30 seconds
